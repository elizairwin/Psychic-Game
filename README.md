# Psychic-Game

## Description: 
A very simple game tests the users psychic abilities. The user types a letter guess and attempts to match the letter the computer is thinking of. Alerts are displayed for wins/losses and the game is reset at the end of a session.

## Technology Used:
* HTML
* CSS
* Bootstrap
* JavaScript
